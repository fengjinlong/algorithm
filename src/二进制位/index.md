### 位运算应用

- 判断奇数偶数

  ```javascript
  10 & 1 = 0
  11 & 1 = 1
  ```

- 交换两个数

  ```
  // 1. lat [a, b] = [b,a]
  // 2. 过度方法
  // 3. 异或 异或运算（^）在两个二进制位不同时返回1，相同时返回0。
  
  let a= 10
  let b = 20
  a ^= b
  b ^= a
  a ^= b
  ```

- 变换符号

  ```javascript
  lat a = 10
  a = ~a + 1 // -10
  ```

- 求绝对值

  ```javascript
  // 1 数字由32位构成，第一位为符号位，所以找到符号位
  1 >> 31 // 0
  -1 >> 31 // -1
  // 2 判断 0 1，是否需要 变换符号 求绝对值
  ```

- 乘 2

  ```
  let a = 10
  a << 1 // 20
  a << 2 // 40
  a << 3 // 80
  ```
